## Azure Infrastructure Solution Area## What's newIn the recent Partner Crucible updates, the following changes were made to the Azure Infrastructure Solution Area:- Added [Azure Well-Architected Framework - Service Guides](https://learn.microsoft.com/en-us/azure/well-architected/service-guides/?product=popular): This framework consists of quality-driven tenets, architectural decision points, and review tools to help solution architects build a solid technical foundation for their workloads. Service guides assist in decision-making for individual Azure components within a workload. Each guide focuses on core features and capabilities necessary for excellence.- Added [Architectures for Oracle database on Azure Virtual Machines](https://learn.microsoft.com/en-us/azure/virtual-machines/workloads/oracle/oracle-reference-architecture): This article covers deploying a highly available Oracle database on Azure, including disaster recovery considerations. The architectures outlined are based on customer deployments and are specific to Oracle Database Enterprise Edition.- Added [E-Business suite on Azure](https://learn.microsoft.com/en-us/azure/virtual-machines/workloads/oracle/deploy-application-oracle-database-azure#e-business-suite-on-azure): This architecture is designed for the deployment of Oracle E-Business Suite (EBS) on Azure, which includes Supply Chain Management and Customer Relationship Management applications. The architecture ensures high availability within a single region.## Deep links- [Azure Well-Architected Framework - Service Guides](https://learn.microsoft.com/en-us/azure/well-architected/service-guides/?product=popular): Comprehensive guides to help with decision-making for Azure components.- [Architectures for Oracle database on Azure Virtual Machines](https://learn.microsoft.com/en-us/azure/virtual-machines/workloads/oracle/oracle-reference-architecture): Guide for deploying Oracle database on Azure.- [E-Business suite on Azure](https://learn.microsoft.com/en-us/azure/virtual-machines/workloads/oracle/deploy-application-oracle-database-azure#e-business-suite-on-azure): Architecture for deploying Oracle E-Business Suite on Azure.
