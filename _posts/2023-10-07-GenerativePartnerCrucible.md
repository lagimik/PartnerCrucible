# Digital Apps and Innovation Solution Area## What's new- Added Realtime Vehicle Tracking with Azure MapsSummary: The Partner Crucible updates include the addition of Realtime Vehicle Tracking with Azure Maps. This solution allows for real-time tracking of vehicles using GPS coordinate data, which is streamed into Azure and displayed on a web application using Azure Maps. It includes IoT Hub, Azure Functions, SignalR, App Service, Azure Maps, and Power BI.## Deep links- [Realtime Vehicle Tracking with Azure Maps](https://github.com/tbecks/Realtime-Vehicle-Tracking): Demo - This solution is intended to simulate a vehicle tracking solution in real-time over a web-based mapping application. Vehicle data consists of GPS coordinate data that is streamed into Azure and presented on a web application using Azure Maps. This pattern can be used for any streaming data you would like to visualize on a web-based map. Includes IoT Hub, Azure Functions, SignalR, App Service, Azure Maps, Power BI. (Source: @tbecks GitHub)
